Installation
============

Dependencies
------------

shapicant requires:

- Python (>= 3.6)
- shap (>= 0.36.0)
- numpy
- pandas
- scikit-learn
- tqdm

For Spark, we also need:

- pyspark (>= 3.0)
- pyarrow

User installation
-----------------

The easiest way to install shapicant is using :code:`pip`

.. code:: bash

    pip install shapicant

or :code:`conda`

.. code:: bash

    conda install -c conda-forge shapicant
